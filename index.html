Course Overview

How to plan the behaviour of a digital product? How to define how each element reacts? How to facilitate human-machine understanding? Interaction Design is the discipline that takes care of it.

The interaction design, as well as usability, information architecture and other disciplines, allow you to have a good user experience.

Interaction design is a practice that will become more important as new technologies progress. It is essential to learn the principles involved in this type of design to create amazing interfaces for your users.
